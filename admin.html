How to Use Photo Manager

1. Add Photos

Click the upload area or drag and drop images to add new photos to your gallery.

2. Add Details

Fill in the title, category, steel type, handle material, length and a short description for each photo.

3. Manage Photos

Edit details or remove photos using the buttons on each photo card.

4. Export Data

Use the Export Data button to get the code needed for your website. The export now includes steel, handle, and length fields.

5. Update Website

Copy the exported data and replace the galleryData array in your main.js file.
